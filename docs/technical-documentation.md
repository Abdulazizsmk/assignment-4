# Technical Documentation — Abdulaziz Saud

## Overview
This project is a lightweight static portfolio demonstrating interactive client-side features for SWE363 Assignment 2.

## Files
- `index.html` — markup
- `css/styles.css` — styles and animations
- `js/script.js` — interactive logic
- `docs/ai-usage-report.md` — AI usage log

## How it works
- Projects are stored in a local JS array (can be replaced with API calls).
- localStorage stores name and theme preferences.
- Fetch example uses `https://api.quotable.io/random`.

## Future improvements
- Connect projects to GitHub API to show live repositories.
- Add server-side form handling for real emails.
